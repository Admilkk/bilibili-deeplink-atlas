# 前缀：av

前缀 `av` 下的全部路由，主要归入“av”相关页面或入口。

- 路由数：`1`

## `bilibili://av/{id}`

::: info 路由说明
- 类型：`native`
- 说明：bilibili 协议 路由，归入“av / id 参数”，类型为 原生。
- 关联入口：`bilibili://video/{id}`
:::

### 参数说明

- `id`：通用业务 ID，具体含义取决于所在路由。
