# 前缀：activity-bilipay

前缀 `activity-bilipay` 下的全部路由，主要归入“Bili 支付”相关页面或入口。

- 路由数：`1`

## `activity://bilipay/wallet`

::: info 路由说明
- 类型：`native`

- 关联入口：`bilibili://bilipay/wallet`、`bilibili://bilipay/bcoin/recharge`
:::
