# 前缀：browser

前缀 `browser` 下的全部路由，主要归入“浏览器”相关页面或入口。

- 路由数：`1`

## `bilibili://browser`

::: info 路由说明
- 类型：`web`
- 说明：bilibili 协议 路由，归入“浏览器”，类型为 网页。
- 关联入口：`activity://main/web`
:::

### 参数说明

- 无显式路径参数。
- `url`：已知的目标网页地址字段，详细说明见 `browser-web` 专题页。
